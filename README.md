# Binge Budget

Binge Budget helps users determine how many television episodes
they can watch within a specified amount of time.

## Features

- Search for a television show
- Retrieve show and episode data from the TVmaze API
- Calculate which episodes fit into the user's available time
- Display episode titles, runtimes, summaries, and images
- Handle invalid searches and API errors

## Technologies

- Node.js
- Express.js
- Axios
- EJS
- HTML
- CSS
- TVmaze API

## Installation

1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open `http://localhost:3000`.

## API

This project uses the TVmaze public API.
